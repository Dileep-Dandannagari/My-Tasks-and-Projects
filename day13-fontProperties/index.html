CSS has several different font properties as shown below
  • font : Its a short hand property, sets all the different properties of element font
  • font-family:The font-family CSS property specifies a prioritized list of one or more font family names
  • font-size: The font-size CSS property sets the size of the font.
  • font-style: The font-style CSS property sets weather a font should be styled with a normal , italic, oblique.
  • font-weight : The font-size CSS property sets the weight( or boldness) of the font.

font-stretch: font-stretch property allows us to make text narrower or wider
