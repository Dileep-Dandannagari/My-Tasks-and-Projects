Position property

Static

The element is positioned according to the normal flow of the document. The top, right, bottom, left, z-index properties have no effect. This is the default value.


Relative

If we set position as relative for an element the element is positioned according to the normal flow of the document. and then we can offset or move element relative to itself by using top, right, bottom and left. The offset does not effect position of any other elements; thus, the space given for the element in the webpage layout is same as if the position were static.


Absolute

The element is removed from the normal flow of document, and no space is created for the element in the page layout. The element is positioned relative to its closest ancestor(if any) or the initinal containing block(body). Its final position is determined by the values of top, right, botto, and left.


Fixed

The element is removed from the normal document flow, and no space is create for the element in the page layout. The element is postioned relative to its initial containing block. It's final position is determined by the values of top, bottom, right, left.


Sticky

The element is positioned according to the normal flow of the document , and then offset relative to its nearest scrolling ancestor and containing block. The offset does not effect the position of any other element
