CSS Specificity :

Specificity comes into play when conflictin styles target the same elements. If the same selector target the same element then order matters. And if the different selectors targeting the same elements then Specificity comes into play CSS Specificity is how the browser decides which rules to apply when multiple rules could apply to the same element . It is the measure of how specific a given selector is. The more specific selector wins.
